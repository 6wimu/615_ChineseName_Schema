Section 05

Sample XML Records

Six complete, validated instances using homepage-style schema previews. Together they demonstrate every mapping type, optional family names, IPA, repeatable radicals, and a Five Elements record.

01. David → 戴伟02. 박다운 → 朴多韵03. Victor → 胜伟04. Sofia → 索菲娅05. Wilson → 韦炜森06. はるか (Haruka) → 春香Combined file

Validation coverage

6 complete recordsEach record is available as a standalone XML file and in one combined database file.
5 mapping typesPhonetic, Semantic, Localization, HanjaSphere, and Phono-Semantic are all represented.
Optional familyNameVictor, Sofia, and Haruka validate without an invented surname.
Five ElementsWilson → 韦炜森 explicitly records 火, 木, and FiveElements.
