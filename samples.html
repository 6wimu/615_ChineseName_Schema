Instance Record 01: Western Name Localization

CN-2026-0002
戴伟 from David (English)
Pinyin
Dài Wěi
IPA Pronunciation
[taɪ̯˥˧ weɪ̯˨˩˦]
Nationality (ISO 3166)
US (United States)
Mapping Strategy
Localization
Hanzi Radical (부수)
亻 (인변)
Difficulty for Foreigners
Low
Semantic Description (의미론적 분석 사유)
원래 서양 이름인 'David'의 첫 발음 소리를 살리면서, 중국의 유명한 전통 성씨인 '대(戴)'씨로 연결했습니다. 위대하고 훌륭한 사람이 되라는 뜻입니다. 
<?xml version="1.0" encoding="UTF-8"?>
<ChineseNameDatabase
    xmlns="http://www.skku.edu/metadata/cn/"
    xmlns:dcterms="http://purl.org/dc/terms/"
    xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
    xsi:schemaLocation="http://www.skku.edu/metadata/cn/ cn_schema.xsd">
    <NameRecord>
        <dcterms:identifier>CN-2026-0002</dcterms:identifier>
        <originalName>David</originalName>
        <inLanguage>English</inLanguage>
        <familyName>戴</familyName>
        <givenName>伟</givenName>
        <fullChineseName>戴伟</fullChineseName>
        <pinyin>Dài Wěi</pinyin>
        <ipa>[taɪ̯˥˧ weɪ̯˨˩˦]</ipa>
        <nationality>US</nationality>
        <mappingType>Localization</mappingType>
        <dcterms:description>원래 서양 이름인 'David'의 첫 발음 소리를 살리면서, 중국의 유명한 전통 성씨인 '대(戴)'씨로 연결했습니다. 위대하고 훌륭한 사람이 되라는 뜻입니다.</dcterms:description>
        <hanziRadical>亻</hanziRadical>
        <pronunciationDifficulty>Low</pronunciationDifficulty>
        <culturalElement>None</culturalElement>
    </NameRecord>
</ChineseNameDatabase>

IPA Pronunciation
[pʰjaʊ̯˧˥ twɔ˥ yn˥˩]

KR (South Korea)

Phono-Semantic

夕, 音 (다중 지정)

Medium
Semantic Description (의미론적 분석 사유)
한국어 고유어 발음과 동일하게 읽히는 한자를 골랐습니다. 아름다운 예술적 운치와 풍요로움이 가득하길 바란다는 뜻입니다.

        <dcterms:identifier>CN-2026-0007</dcterms:identifier>

        <ipa>[pʰjaʊ̯˧˥ twɔ˥ yn˥˩]</ipa>
        <nationality>KR</nationality>
        <mappingType>Phono-Semantic</mappingType>
        <dcterms:description>한국어 고유어 발음과 동일하게 읽히는 한자를 골랐습니다. 아름다운 예술적 운치와 풍요로움이 가득하길 바란다는 뜻입니다.</dcterms:description>

        <culturalElement>None</culturalElement>
    </NameRecord>
