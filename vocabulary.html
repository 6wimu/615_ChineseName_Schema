Complete data dictionary

dcterms:identifier — Record ID

Source: Dublin Core Terms

Definition: Unique identifier for each recommendation record.

Data value: string

Cardinality: Required · single

Example: CN-2026-0011

originalName — Original Name

Source: Local

Definition: Complete name before Chinese-name mapping.

Data value: string

Cardinality: Required · single

Example: Victor

inLanguage — Original Name Language

Source: Local controlled vocabulary

Definition: Language of the original name.

Data value: English / Korean / Japanese / Other

Cardinality: Required · single

Example: English

familyName — Chinese Family Name

Source: Local; informed by FOAF

Definition: Chinese surname component. It is omitted when the recommendation has no conventional family name.

Data value: string

Cardinality: Optional · single

Example: 戴; omitted in Victor → 胜伟

givenName — Chinese Given Name

Source: Local; informed by FOAF

Definition: Chinese personal-name component. When familyName is absent, it stores the complete recommended personal name.

Data value: string

Cardinality: Required · single

Example: 胜伟

fullChineseName — Full Chinese Name

Source: Local

Definition: Complete Chinese name used for display and retrieval.

Data value: string

Cardinality: Required · single

Example: 胜伟

pinyin — Pinyin

Source: Local

Definition: Mandarin pronunciation in Hanyu Pinyin with tone marks.

Data value: string

Cardinality: Required · single

Example: Shèng Wěi

ipa — IPA Pronunciation

Source: Local

Definition: Optional International Phonetic Alphabet transcription of the Mandarin pronunciation.

Data value: string

Cardinality: Optional · single

Example: [ʂɤŋ˥˩ weɪ̯˨˩˦]

nationality — Nationality

Source: Local; ISO 3166-1

Definition: Two-uppercase-letter ISO 3166-1 alpha-2 lexical format.

Data value: pattern [A-Z]{2}

Cardinality: Required · single

Example: CA

mappingType — Name Mapping Type

Source: Local controlled vocabulary

Definition: Principal strategy used to create or preserve the Chinese name.

Data value: Phonetic / Semantic / Localization / HanjaSphere / Phono-Semantic

Cardinality: Required · single

Example: Semantic

dcterms:description — Name Meaning and Rationale

Source: Dublin Core Terms

Definition: Meaning of the selected characters and the recommendation rationale.

Data value: string

Cardinality: Required · single

Example: Character meaning and design explanation

hanziRadical — Hanzi Radical

Source: Local

Definition: Repeatable radical values for key given-name characters. Values such as 火 and 木 can expose a Five Elements rationale.

Data value: string

Cardinality: Optional · repeatable

Example: 火; 木

pronunciationDifficulty — Pronunciation Difficulty

Source: Local controlled vocabulary

Definition: Estimated Mandarin-pronunciation difficulty for non-Chinese speakers.

Data value: Low / Medium / High

Cardinality: Optional · single

Example: Medium

culturalElement — Cultural Element

Source: Local controlled vocabulary

Definition: Cultural or philosophical consideration used in the recommendation.

Data value: Zodiac / FiveElements / None

Cardinality: Optional · repeatable

Example: FiveElements
